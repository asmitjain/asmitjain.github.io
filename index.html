AREA OF INTEREST

Having skills in the mechanical design along with the knowledge of manufacturing practices, is a great combination. It helps you to make an idea into reality, which is one of the most crucial skills I possess. It bolsters my confidence to incline more towards these two domains. A combination of these skills helps me to work as a cross-functional team member and discuss feasibility and technicalities with colleagues/ vendors/ suppliers, utilize my resources for traditional and advanced manufacturing methods, and work around the proposed project budgets in a given timeframe.

The designer should not only design highly sophisticated components but also needs to consider what the manufacturing and quality process will be while in the designing phase. All these things need to be done when technologies are evolving and changing every year. Considering this, I have developed some set of skills which help me to:

-> Design any parts from scratch, perform design analysis by simulating real-life conditions and optimize the components for enhancing the performance.
-> Develop prototypes, pilot devices, tools & fixtures in less turnaround time, using subtractive/ additive manufacturing techniques.
-> Transition a new part from the prototyping phase to mass production, set the processes, schedules, SOPs for better efficiencies.
-> Automate the process by python or PLC/ HMI for large-scale, stand-alone production systems.
-> Implement data acquisition process which will acquire data in real-time and can plot graphs to monitor the operations.
-> Get the quality parts by implementing six sigma/ lean principles, APQP, PPAP, DVP&R, DFM/A, etc. and electrical and safety certifications like UL, CE, GMark, SAA, etc.
